# Local AI voice input Browser Extension

This is a basic Chrome extension that uses a local voice service to interact with various AI websites

## Features
- Supports ChatGPT and Copilot


## Installation
1. Download the zip file of this repository. Select from Code button above or [Click here](https://github.com/js-rocket-org/webext-local-ai-voice/archive/refs/heads/main.zip)
2. Extract zip file to a folder
3. Open Chrome and go to `chrome://extensions`
4. Enable "Developer mode"
5. Click "Load unpacked" and select the folder (which should contain the `manifest.json` file)


## File Structure
- `manifest.json`: Extension manifest
- `content.js`: Content script
